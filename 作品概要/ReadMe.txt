作品概要：
//DirectX11,c++を使用し、スーパーマリオブラザーズ を参考に、少し要素を足し制作したランゲームです。
//薪(アイテム)を集めてジェットパックを使用し空を飛びゴールにたどり着くためにステージを進んでいきます。
//ジェットパックを使用すると薪を消費します。
//薪は最終のスコアにかかわるため、なるべく残した状態でクリアすることが理想です。

開発環境：
//ハードウェア：WindowsPC
//開発用OS：Windows11,Windows10
//統合開発環境： Microsoft Visual Studio 2022
//開発言語：C++ / HLSL
//使用ライブラリ：授業で作成したDirectX11を用いたエンジン / FBXSDK

操作説明：
//入力デバイス：キーボード
//スペースキー：ジャンプ/ゲーム進行/ジャンプ中にもう一度スペースキーでジェットパック使用
//Aキー：左方向に移動
//Bキー：右方向に移動
